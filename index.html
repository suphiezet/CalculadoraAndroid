🧮 Sobre a Calculadora

A Calculadora Bonita é um aplicativo Android nativo, desenvolvido em Java puro, sem dependências externas e sem arquivos XML de layout (exceto o manifesto). Ela oferece uma experiência visual moderna, com gradiente azul e vermelho, botões arredondados, e uma lógica completa de avaliação de expressões matemáticas.

🔹 Operações básicas: adição, subtração, multiplicação, divisão.
🔹 Funcionalidades extras: inversão de sinal (±), backspace, porcentagem, limpar tudo.
🔹 Interface adaptativa: usa pesos para se ajustar a qualquer tamanho de tela.
🔹 Ícone adaptativo: criado com vetores XML, seguindo as diretrizes do Material Design.

O código foi compilado manualmente com as ferramentas de build do Android SDK (aapt2, d8, zipalign, apksigner), sem uso do Gradle, garantindo total controle sobre o processo.

⚙️ Como eu criei

O processo foi totalmente artesanal. Comecei com um arquivo Java simples e um `AndroidManifest.xml`, e fui construindo o APK passo a passo: 

1. Compilação Java → javac -cp android.jar -d obj src/com/exemplo/calculadora/*.java
2. Conversão para DEX → d8 --lib android.jar --output bin/ obj/...
3. Compilação de recursos (ícones) → aapt2 compile --dir res -o obj/compiled_res
4. Geração do APK base → aapt2 link -o bin/app.unaligned.apk -I android.jar --manifest AndroidManifest.xml -R obj/compiled_res
5. Inserção do DEX → jar uf bin/app.unaligned.apk classes.dex
6. Alinhamento → zipalign -p -f 4 bin/app.unaligned.apk bin/app.aligned.apk
7. Assinatura → apksigner sign --ks debug.keystore ... --out bin/CalculadoraBonita.apk bin/app.aligned.apk

Todo esse fluxo foi automatizado com um script build.bat para Windows. O resultado é um APK funcional que roda em dispositivos Android 5.0+.
